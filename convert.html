tasks': Google Tasks made beautiful

Sometimes you just want to see your tasks all at once, with no distractions. That's what tasks' does for you: all your Google Tasks in one place, with a simple, beautiful interface.

Start using tasks' now.

Mildly important fine print: we never store any personal information on our servers; all information is retrieved and stored client-side using the Google Tasks Javascript API. tasks' is not responsible for any loss of data caused by the use of tasks', although we promise to be very careful. A popup will appear the first time you use tasks' to authenticate with Google; tasks' will never show any popups other than that. tasks' will never make changes to your task lists without user input. tasks' was made by Will Haldean Brown.
